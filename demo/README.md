# Skeletonify Demo

Interactive playground showing Skeletonify in action.

```bash
cd demo
npm install vite @vitejs/plugin-react tailwindcss postcss autoprefixer
npm run dev
```

Or from the repo root:

```bash
npx vite demo
```

## What's inside

- **Hero** — animated title + one-line pitch.
- **Interactive playground** — toggle `Loading ON / OFF` and watch every card switch between real UI and inferred skeleton with zero flicker.
- **Four real-world examples** — Profile Card, Dashboard Card, Table Row List, Blog Post.
- **Without vs With comparison** — side-by-side of the spinner-era loading state vs Skeletonify.
- **Modern SaaS aesthetic** — dark mode, soft gradients, clean typography, subtle motion.

The demo imports directly from `../src` so any change to Skeletonify itself is reflected live.
